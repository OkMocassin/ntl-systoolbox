# Gamma.app Presentation Prompts

Copy and paste the following prompt directly into **Gamma.app** (using the "Generate from Text" option) to automatically generate a beautiful, structured presentation about your project.

---

### Prompt for Gamma.app:

```text
Please generate a professional, modern, and engaging technical presentation for an IT administration tool called "NTL System Toolbox". 

Audience: IT Managers and System Administrators. 
Tone: Professional, innovative, and clear.

Please structure the presentation with the following slides:

1. Title Slide: NTL System Toolbox
- Subtitle: A comprehensive interactive CLI for System Diagnostics, Backups, and Obsolescence Auditing.

2. The Problem
- Modern IT infrastructure is fragmented.
- Sysadmins juggle dozens of scripts, manual terminal commands, and unconnected tools to check server health, run backups, and audit networks. 
- Human error is high when typing complex command-line arguments.

3. The Solution: NTL System Toolbox
- An interactive, menu-driven Python CLI.
- Consolidates Diagnostics, MySQL Backups, and Network Auditing into one centralized hub.
- Color-coded, highly visual terminal output that highlights critical issues instantly.

4. Core Architecture
- Built on Python 3.9+.
- Modular design: separate modules for Diagnostics, Backups, and Obsolescence.
- Configurable via simple YAML files, allowing easy deployment.

5. Key Feature 1: Health Diagnostics
- Uses cross-platform libraries to monitor CPU, RAM, and Disk utilization.
- Monitors critical services (like Active Directory and DNS).
- Flags critical issues and links instantly to Remediation Runbooks.

6. Key Feature 2: Automated Backups
- Integrates directly with MySQL databases.
- Performs full database dumps, supports dry-runs, and cleans up historical, outdated backups to save space.

7. Key Feature 3: Obsolescence Auditing
- Uses Nmap to discover hardware and OS versions across the network.
- Identifying end-of-life (EOL) devices before they become security vulnerabilities.

8. Why This Tech Stack? (The "Secret Sauce")
- Python: Rapid development and unmatched automation ecosystem.
- Rich & Questionary: Replaces confusing CLI flags with interactive menus and beautiful terminal tables.
- Pydantic: Ensures flawless data validation to prevent crashes.

9. Future Roadmap
- Deeper Active Directory LDAP Integration.
- Cloud Backup targets (AWS S3/Azure Blob).
- Automated weekly Slack/Teams reporting.

10. Conclusion & Q&A
- Wrap up summarizing how it saves time, reduces human error, and centralizes administrative visibility.
```

---

### Instructions for best results in Gamma:
1. Go to gamma.app and click **"Generate"**.
2. Select **"Text to Deck"** (or paste text).
3. Paste the prompt above.
4. Choose a "Dark" or "Cyber/Tech" theme that fits a terminal/CLI aesthetic for the best visual impact.
